About Steps Playground

Steps Playground is a browser-based Steps learning environment that uses a subset of the full Steps language as an introduction.

The playground focuses on one visible step: main, core statements, expressions, and simple riser helpers so you can learn the basics without full project setup.

What is missing here

  • Visible buildings and floors
  • Full project structure and multi-file workflow
  • Some full-product and non-browser-friendly features
  • See "About Steps Language" for full language details.

Examples are intended for watching what happens. Tutorials are intended to help you change code and make something happen yourself.
